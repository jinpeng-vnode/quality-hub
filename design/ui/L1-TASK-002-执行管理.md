## 组件列表

| 组件 | Ant Design 组件 | 数据来源 | 说明 |
|------|-----------------|----------|------|
| 面包屑 | `a-breadcrumb` | 路由参数 + store | — |
| 触发执行按钮 | `a-button type="primary"` | — | 打开触发执行配置弹窗 |
| 状态筛选 | `a-select` | 本地枚举 | 全部/pending/running/passed/failed/error |
| 执行记录表格 | `a-table` | API: GET /api/runs?projectId=x | — |
| 状态标签 | `a-tag` | 行数据 | pending=默认, running=蓝色+旋转图标, passed=绿色, failed=红色, error=橙色 |
| 通过率 | `a-progress` | 行数据 passed/total | 百分比展示 |
| 操作列 | `a-button` | — | 查看详情 |
| 触发执行弹窗 | `a-modal` + `a-form` | 本地状态 | 选择要执行的用例 |
| 详情页概要 | `a-descriptions` | API: GET /api/runs/:id | 展示执行元信息 |
| 结果表格 | `a-table` | API: GET /api/runs/:id/results | 每条用例的执行结果 |
| 错误详情展开 | `a-table` expandable | 行数据 errorMessage | 展开行显示错误信息和截图 |
| 截图预览 | `a-image` | 行数据 screenshotUrl | 失败用例的截图 |
| 分页器 | `a-pagination`（表格内置） | 前端分页 | 默认 pageSize=20 |

## 交互行为

| 用户操作 | 触发事件 | 页面响应 |
|----------|----------|----------|
| 点击「触发执行」 | 打开 Modal | 显示执行配置表单 |
| 选择用例并提交 | POST /api/runs | 关闭弹窗，新执行记录出现在表格顶部（status=pending） |
| 选择状态筛选 | 前端过滤 | 表格按状态过滤 |
| 点击行「查看详情」 | 路由跳转 | 进入执行详情页 `/projects/:id/runs/:runId` |
| 执行进行中 | 轮询 GET /api/runs/:id（每 5s） | 状态实时更新直到终态 |
| 在详情页点击失败用例行 | 展开行 | 显示 errorMessage 和 screenshotUrl |
| 切换分页 | 前端分页 | 表格更新 |

## 状态覆盖

| 状态 | 表现 |
|------|------|
| 加载中 | 表格显示 `loading` 旋转指示器 |
| 空数据 | `a-empty`，描述「暂无执行记录，点击上方按钮触发执行」 |
| 错误 | `a-alert type="error"` + 重试按钮 |
| 正常 | 表格正常展示执行记录 |

## 表单字段（触发执行）

| 字段 | 类型 | 组件 | 校验规则 | 说明 |
|------|------|------|----------|------|
| caseIds | number[] | `a-transfer` | 必填，至少选 1 个 | 穿梭框选择要执行的用例（仅 caseType=e2e 的用例） |
| envUrl | string | `a-input` | 可选，URL 格式 | 目标环境 URL |

## 调用的后端 API

| API | 方法 | 触发时机 | 用途 |
|-----|------|----------|------|
| `/api/runs?projectId={id}` | GET | 页面加载 | 获取执行记录列表 |
| `/api/runs` | POST | 提交触发执行表单 | 创建新执行任务（body 含 projectId, caseIds） |
| `/api/runs/{runId}` | GET | 进入详情页、轮询 | 获取执行详情 |
| `/api/runs/{runId}/results` | GET | 详情页加载 | 获取用例执行结果列表 |
| `/api/cases?featureId={id}` | GET | 打开触发弹窗 | 获取可选用例列表（穿梭框数据源） |
| `/api/features?projectId={id}` | GET | 打开触发弹窗 | 获取功能点列表（辅助筛选用例） |
