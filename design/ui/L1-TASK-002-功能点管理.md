# L1-TASK-002 功能点管理页面 UI 规格

## 页面信息

| 属性 | 值 |
|------|-----|
| 路由 | `/projects/:projectId/features` |
| 文件路径 | `src/views/FeatureListView.vue` |
| 权限 | 登录用户（V1 无认证） |
| 导航位置 | 左侧菜单第 2 项，图标 `AppstoreOutlined` |

## 页面布局


## 组件列表

| 组件 | Ant Design 组件 | 数据来源 | 说明 |
|------|-----------------|----------|------|
| 面包屑 | `a-breadcrumb` | 路由参数 + store | 显示项目名称层级 |
| 新增按钮 | `a-button type="primary"` | — | 打开新增功能点弹窗 |
| 状态筛选 | `a-select` | 本地枚举 | 选项：全部/待覆盖(pending)/部分覆盖(partial)/已覆盖(covered) |
| 搜索框 | `a-input-search` | 本地状态 | 按功能点标题前端过滤 |
| 功能点表格 | `a-table` | API: GET /api/features?projectId=x | 展示功能点列表 |
| 状态标签 | `a-tag` | 行数据 | pending=默认/partial=蓝色/covered=绿色 |
| 用例数 | 文本 | 行数据 caseCount | 可点击跳转到用例列表 |
| 操作列 | `a-space` + `a-button` | — | 编辑、删除 |
| 新增/编辑弹窗 | `a-modal` + `a-form` | 本地状态 | — |
| 分页器 | `a-pagination`（表格内置） | API 返回 total | 默认 pageSize=20 |

## 交互行为

| 用户操作 | 触发事件 | 页面响应 |
|----------|----------|----------|
| 点击「新增功能点」 | 打开 Modal | 显示新增表单弹窗 |
| 填写表单并提交 | POST /api/features | 关闭弹窗，刷新表格，成功提示 |
| 选择状态筛选 | 前端过滤 | 表格按状态过滤 |
| 输入搜索关键词 | 前端过滤 | 表格按标题过滤，防抖 300ms |
| 点击行「编辑」 | 打开 Modal | 回填数据的编辑表单 |
| 点击行「删除」 | `Modal.confirm` | 确认后 DELETE，刷新表格；若返回 409 则提示「该功能点下还有用例，无法删除」 |
| 点击用例数 | 路由跳转 | 跳转到测试用例页面 `/projects/:id/cases?featureId=x` |
| 切换分页 | 前端分页 | 表格更新 |

## 状态覆盖

| 状态 | 表现 |
|------|------|
| 加载中 | 表格显示 `loading` 属性的旋转指示器 |
| 空数据 | 表格内嵌 `a-empty`，描述「暂无功能点，点击上方按钮新增」 |
| 错误 | `a-alert type="error"` 显示在表格上方 + 重试按钮 |
| 正常 | 表格正常展示数据 |

## 表单字段（新增/编辑功能点）

| 字段 | 类型 | 组件 | 校验规则 | 说明 |
|------|------|------|----------|------|
| title | string | `a-input` | 必填，1-200 字符 | 功能点标题 |
| description | string | `a-textarea` | 可选 | 功能点描述 |
| source | enum | `a-select` | 可选 | 选项：prd/issue/manual |

## 调用的后端 API

| API | 方法 | 触发时机 | 用途 |
|-----|------|----------|------|
| `/api/projects/{id}` | GET | 页面加载 | 获取项目名称（面包屑） |
| `/api/features?projectId={id}` | GET | 页面加载 | 获取功能点列表 |
| `/api/features` | POST | 提交新增表单 | 创建功能点（body 含 projectId） |
| `/api/features/{fid}` | PUT | 提交编辑表单 | 更新功能点 |
| `/api/features/{fid}` | DELETE | 确认删除 | 删除功能点 |
