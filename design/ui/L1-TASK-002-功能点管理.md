# L1-TASK-002 功能点管理页面 UI 规格

## 页面信息

| 属性 | 值 |
|------|-----|
| 路由 | `/projects/:projectId/features` |
| 文件路径 | `src/views/features/FeatureList.vue` |
| 权限 | 登录用户，需属于当前项目 |
| 导航位置 | 左侧菜单第 2 项，图标 `AppstoreOutlined` |

## 页面布局

## 组件列表

| 组件 | Ant Design 组件 | 数据来源 | 说明 |
|------|-----------------|----------|------|
| 面包屑 | `a-breadcrumb` | 路由参数 + store | 显示项目名称层级 |
| 新增按钮 | `a-button type="primary"` | — | 打开新增功能点弹窗 |
| 批量导入按钮 | `a-button` | — | 打开导入弹窗（支持从 PRD 解析） |
| 状态筛选 | `a-select` | 本地枚举 | 选项：全部/待开发/开发中/已完成/已废弃 |
| 搜索框 | `a-input-search` | 本地状态 | 按功能点名称模糊搜索 |
| 功能点表格 | `a-table` | API: GET /api/projects/:id/features | 展示功能点列表 |
| 状态标签 | `a-tag` | 行数据 | 不同状态不同颜色 |
| 覆盖率进度 | `a-progress` | 行数据 | 用例覆盖率百分比 |
| 操作列 | `a-space` + `a-button` | — | 编辑、删除、查看用例 |
| 新增/编辑弹窗 | `a-modal` + `a-form` | 本地状态 | — |
| 分页器 | `a-pagination`（表格内置） | API 返回 total | 默认 pageSize=20 |

## 交互行为

| 用户操作 | 触发事件 | 页面响应 |
|----------|----------|----------|
| 点击「新增功能点」 | 打开 Modal | 显示新增表单弹窗 |
| 填写表单并提交 | POST /api/projects/:id/features | 关闭弹窗，刷新表格，成功提示 |
| 选择状态筛选 | GET /api/...?status=xxx | 表格按状态过滤 |
| 输入搜索关键词 | GET /api/...?keyword=xxx | 表格按关键词过滤，防抖 300ms |
| 点击行「编辑」 | 打开 Modal | 回填数据的编辑表单 |
| 点击行「删除」 | 弹出确认框 | 确认后 DELETE，刷新表格 |
| 点击行「查看用例」 | 路由跳转 | 跳转到测试用例页面并过滤该功能点 |
| 点击「批量导入」 | 打开 Modal | 显示文本输入区域，粘贴功能点列表 |
| 切换分页 | GET /api/...?page=x | 表格更新 |

## 状态覆盖

| 状态 | 表现 |
|------|------|
| 加载中 | 表格显示 `loading` 属性的旋转指示器 |
| 空数据 | 表格内嵌 `a-empty`，描述「暂无功能点，点击上方按钮新增」 |
| 错误 | `a-alert type="error"` 显示在表格上方 + 重试按钮 |
| 正常 | 表格正常展示数据 |

## 表单字段（新增/编辑功能点）

| 字段 | 类型 | 组件 | 校验规则 | 说明 |
|------|------|------|----------|------|
| 功能点名称 | string | `a-input` | 必填，2-100 字符 | — |
| 描述 | string | `a-textarea` | 可选，最多 500 字符 | — |
| 来源 | enum | `a-select` | 必填 | 选项：PRD/需求评审/线上反馈/自定义 |
| 优先级 | enum | `a-select` | 必填 | P0/P1/P2/P3 |
| 状态 | enum | `a-select` | 必填 | 待开发/开发中/已完成/已废弃 |

## 调用的后端 API

| API | 方法 | 触发时机 | 用途 |
|-----|------|----------|------|
| `/api/projects/:id/features` | GET | 页面加载、筛选、翻页 | 获取功能点列表 |
| `/api/projects/:id/features` | POST | 提交新增表单 | 创建功能点 |
| `/api/projects/:id/features/:fid` | PUT | 提交编辑表单 | 更新功能点 |
| `/api/projects/:id/features/:fid` | DELETE | 确认删除 | 删除功能点 |
| `/api/projects/:id/features/import` | POST | 批量导入提交 | 批量创建功能点 |
