## 二、项目管理 API（Issue #5）

### TC-010 [Type-A] [项目] 创建项目 — 正常流程
- API: `POST /api/projects` body: `{"name":"测试项目","description":"描述","repoUrl":"https://github.com/test/repo"}`
- 断言: 返回 200/201，响应含 `id`、`name`、`createdAt`

### TC-011 [Type-A] [项目] 创建项目 — 名称为空时返回校验错误
- API: `POST /api/projects` body: `{"name":""}`
- 断言: 返回 422

### TC-012 [Type-A] [项目] 创建项目 — 名称重复时返回错误
- API: 先创建项目A，再用相同名称创建
- 断言: 第二次返回 409 或含错误信息

### TC-013 [Type-A] [项目] 获取项目列表
- API: `GET /api/projects`
- 断言: 返回 200，响应为数组

### TC-014 [Type-A] [项目] 获取项目详情
- API: `GET /api/projects/{id}`
- 断言: 返回 200，响应含 `id`、`name`、`repoUrl`

### TC-015 [Type-A] [项目] 获取不存在的项目返回 404
- API: `GET /api/projects/99999`
- 断言: 返回 404，响应含 `code: "NOT_FOUND"`

### TC-016 [Type-A] [项目] 删除项目
- API: `DELETE /api/projects/{id}`
- 断言: 返回 200，响应含 `ok: true`

### TC-017 [Type-A] [项目] 删除不存在的项目返回 404
- API: `DELETE /api/projects/99999`
- 断言: 返回 404
