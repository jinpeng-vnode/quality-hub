# quality-hub
项目质量管理平台 — 功能点追踪、测试用例管理、Midscene E2E 执行、质量报告看板

🌐 线上地址：https://quality-hub.todonot.com

---

## 功能全景图

```
quality-hub
├── 📁 项目管理 ✅
│   ├── 项目 CRUD ✅
│   ├── 项目卡片列表 ✅
│   └── 级联删除（项目→功能点→用例→执行记录） ✅
├── 📁 功能点管理 ✅
│   ├── 功能点 CRUD ✅
│   ├── 按项目筛选 ✅
│   └── 级联删除（功能点→关联用例） ✅
├── 📁 测试用例管理 ✅
│   ├── 用例 CRUD ✅
│   ├── 批量创建用例 ✅
│   ├── 按功能点筛选 / 全量查询 ✅
│   ├── 用例类型（手动 / E2E） ✅
│   └── Midscene 脚本字段 ✅
├── 📁 执行管理 ✅
│   ├── 触发执行（全项目 / 指定用例） ✅
│   ├── 执行记录列表 ✅
```
